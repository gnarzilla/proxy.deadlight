# Contributing to Deadlight

Thanks for your interest! All contributions are welcome — bug reports, docs, code, coffee.

## Quick start
1. Fork → create branch → make changes
2. Run tests (if present) or at least `npm run dev` / binary locally
3. Open PR with clear title and description

## Code style
- JavaScript: Prettier default
- C: `clang-format -i` (style in .clang-format)

## Signing commits
Not required, but DCO sign-off is appreciated:
`Signed-off-by: Name <email>`

Questions? Open an issue — I usually reply within a day.
